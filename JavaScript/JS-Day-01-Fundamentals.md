# JavaScript Day 01: Fundamentals

## Topics Covered
- Variables (`var`, `let`, `const`)
- Data types
- Operators
- Type conversion

## Examples
```js
let name = "Nidhi";
const age = 21;
console.log(name, age);
```

## Practice
- Write 5 variable declarations using different types
- Compare `==` vs `===`

## Key Takeaways
- Prefer `let` and `const` over `var`
- JavaScript is dynamically typed
