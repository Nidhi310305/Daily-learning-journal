# JavaScript Day 02: Functions

## Topics Covered
- Function declaration
- Function expression
- Arrow functions
- Parameters and return values

## Examples
```js
function add(a, b) {
  return a + b;
}

const multiply = (a, b) => a * b;
```

## Practice
- Write functions for sum, difference, and product
- Convert a normal function to arrow function

## Key Takeaways
- Functions make code reusable and modular
- Arrow functions have shorter syntax
